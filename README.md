# EeveePython

EeveePython is a custom Python variant with a unique file extension (`.epy`) for all EeveePython coding files. This project aims to create an enhanced Python development experience with a distinctive identity.

## Features

- Custom `.epy` file extension for all Python code files
- Full Python compatibility
- Easy-to-identify project files

## File Structure

All Python code in this project uses the `.epy` extension:
- `*.epy` - EeveePython source files

## Getting Started

### Installation

1. Clone this repository:
```bash
git clone https://github.com/pacifichatvr6755-blip/EeveePython.git
cd EeveePython
```

2. Set up your environment to recognize `.epy` files

### Usage

Create `.epy` files and use them like standard Python files:

```epy
# hello.epy
print("Welcome to EeveePython!")
```

## Development

To develop with EeveePython:

1. Write your code in `.epy` files
2. Execute with Python interpreter: `python file.epy`
3. All core functionality should use `.epy` extension
